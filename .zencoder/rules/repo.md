# rumah_jurnal Information

## Summary
A Next.js project bootstrapped with `create-next-app`, designed as a web application named `rumah_jurnal`. It utilizes the Next.js App Router and Tailwind CSS for styling.

## Structure
- **src/app/**: Main application code using Next.js App Router.
- **src/lib/**: Library code and utilities.
- **public/**: Static assets (images, icons).
- **.next/**: Generated build output.

## Language & Runtime
**Language**: TypeScript
**Version**: Node.js (Next.js 15.5.4)
**Build System**: Next.js CLI
**Package Manager**: npm

## Dependencies
**Main Dependencies**:
- next: 15.5.4
- react: 19.1.0
- react-dom: 19.1.0
- mysql2: ^3.15.3
- bcryptjs: ^3.0.3
- jsonwebtoken: ^9.0.3

**Development Dependencies**:
- typescript: ^5
- tailwindcss: ^4
- eslint: ^9

## Build & Installation
```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm run start

# Lint code
npm run lint
```
